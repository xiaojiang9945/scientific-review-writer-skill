# Release Notes

## v1.0.0

Initial release package for the `scientific-review-writer` Codex skill.

### Scope

- General scientific review writing skill, not limited to plants.
- English by default unless the user requests another language.
- Full-review default scale: 6,000-9,000 body words, 6 main figures, at least 58 references, and 5 QC rounds.
- Demo based on the cytoskeleton/cell-division review case.

### Release Checks

- Demo manuscript text manually reviewed for workflow notes, prompt traces, claim-boundary labels, and process language.
- Six demo figures visually inspected for white background, spacing, legibility, and non-overlap.
- Three manuscript tables checked in Markdown and DOCX preview.
- Nature-family reference style normalized and validated.
- Markdown validator, DOCX audit, skill quick validator, package build, and privacy scan passed.

### Visibility

The repository can remain private after this release package. Changing repository visibility to public remains a separate owner decision.
