# Plant Kinetochore Review Demo

This private demo uses a plant kinetochore review manuscript from the `Seed_Vigor_Review` workstream.

## Files

- `input/source_review_en_strict.md`: source manuscript draft used as the demo input.
- `output/plant_kinetochore_review_demo.md`: refined review output.
- `output/figures/`: generated review figures referenced by the output manuscript.
- `qc/demo_qc_public.md`: sanitized QC record for repository validation.

## What This Demo Shows

- A formal review manuscript with title, abstract, keywords, main text, figures, captions, outstanding questions, and references.
- Figure references embedded in the result section where the figure is discussed.
- A plant-science review arc that moves from measurement boundaries to mechanism, species diversity, engineering, and outlook.
- A QC trail that documents literature intake, citation/format checks, logic/language review, figure review, and privacy checks.

## What Is Intentionally Excluded

- Raw downloaded full-text files.
- Large literature metadata exports.
- Local absolute paths.
- Private workspace notes not needed to understand the demo.
