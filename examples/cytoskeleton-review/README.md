# Cytoskeleton Review Demo

This demo uses the latest cell-division/cytoskeleton review standard from the local review workstream.

## Files

- `input/source_review_en_final.md`: earlier English manuscript used as demo input.
- `output/cytoskeleton_review_demo.md`: deepened review output.
- `output/figures/`: six generated review figures referenced by the output manuscript.
- `qc/demo_qc_public.md`: sanitized QC record for repository validation.

## What This Demo Shows

- English review manuscript by default.
- Abstract of about 180-220 words.
- Keywords below the abstract.
- Body length within the 6,000-9,000 word full-review range.
- Six manuscript-linked figures with captions below images.
- At least 58 references normalized to a Nature-family numbered style.
- Five-round QC.

## What Is Excluded

- Raw downloaded full-text files.
- Large literature metadata exports.
- Local absolute paths.
- Private workspace notes not needed for evaluating the demo.
