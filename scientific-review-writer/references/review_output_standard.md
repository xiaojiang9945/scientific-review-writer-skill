# Review Output Standard

Use target-journal instructions first. If the user requests a complete figure-rich scientific review and gives no stricter target, use the default scale derived from the latest cytoskeleton-review case.

## Default Scale

| Component | Default target |
|---|---:|
| Language | English |
| Abstract | 180-220 words |
| Keywords | 6-8 |
| Body before references | 6,000-9,000 words |
| Main figures | 6 |
| References | at least 58 |
| Outstanding questions | about 8 |
| QC rounds | 5 |

## Manuscript Shape

1. Title.
2. Abstract.
3. Keywords.
4. Introduction with thesis and evidence boundary.
5. Conceptual or measurement framework.
6. Mechanistic or causal sections.
7. Diversity, exceptions, systems, methods, or translational implications.
8. Outlook or conclusion.
9. Outstanding Questions, when appropriate.
10. References.

## Formal Output Rules

- Do not include workflow notes in the manuscript.
- Do not include claim-boundary labels or prompt traces.
- Do not include internal checks inside figure captions.
- Put figure captions below figures.
- Cite figures in the text where they are discussed.
- Keep citations close to supported claims.
- State access limits in a QC note or methods-style note, not as unsupported manuscript claims.

## DOCX Expectations

When creating Word output:

- Embed figures rather than linking to absolute paths.
- Preserve image aspect ratios.
- Check embedded media count.
- Check caption numbering and reference heading.
- Use readable body text and smaller reference text.
- Inspect the final document before delivery when possible.
