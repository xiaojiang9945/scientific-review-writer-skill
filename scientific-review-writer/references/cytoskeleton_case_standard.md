This standard is extracted from the latest cell-division/cytoskeleton review demo used in this repository. It sets the default target for complete, figure-rich review manuscripts when no other target is specified.

## Reusable Lessons

- Treat the review as a synthesis problem, not a chronological history.
- Use a central conceptual frame that can organize exceptions.
- Separate measurement layers so claims do not collapse distinct mechanisms into one endpoint.
- Add a dedicated section for exceptions when they change the model.
- Use figures as conceptual checkpoints: framework, constraints, memory, execution, asymmetric cases, and exceptional modes.
- Keep the formal manuscript free of QC narration and generation-process text.
- Use 6,000-9,000 words as the normal range for a 5-8 theme full review unless the target journal specifies otherwise.
