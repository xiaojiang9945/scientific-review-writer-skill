# Structure And Style

## Default Language

Write the review manuscript in English unless the user explicitly asks for another language.

## Abstract

A strong abstract should include:

1. The field problem or unresolved tension.
2. The conceptual shift or synthesis frame.
3. The scope of evidence.
4. The key mechanism, model, method, or decision axis.
5. The remaining bottleneck.

Default length: 180-220 words for full reviews unless the target journal specifies otherwise.

## Keywords

Use 6-8 keywords by default for full reviews. Use the target journal's count when specified.

## Section Rhythm

- Open sections with a claim or tension.
- Define terms when ambiguity changes interpretation.
- Move from evidence to mechanism, boundary, and implication.
- Avoid paper-by-paper listing unless the section is explicitly historical.
- Use transitions that explain why the next section follows.

## Language Quality

Avoid:

- workflow notes,
- prompt traces,
- claim-boundary labels,
- reviewer-response phrasing inside the manuscript,
- unsupported certainty,
- generic final sentences such as "more research is needed" without a specific research design or bottleneck.

Prefer evidence-ranked verbs:

- Direct evidence: shows, demonstrates, quantifies, maps, resolves.
- Strong but incomplete evidence: supports, suggests, is consistent with, nominates.
- Synthesis: couples, constrains, buffers, coordinates, tunes, partitions, rewires.
