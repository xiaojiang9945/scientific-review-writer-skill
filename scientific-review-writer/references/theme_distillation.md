# Theme Distillation

Use this before drafting a full review.

## Title Compression

Draft 3-6 candidate titles before the outline.

Good review titles:

- foreground the core topic,
- expose the synthesis angle or unresolved tension,
- avoid filler such as "a comprehensive review" unless required,
- usually stay within 8-16 words,
- use a precise subtitle only when it adds real meaning.

Reject titles that are broad enough to fit many fields.

## Keyword And Full-Text Gate

Before main drafting:

1. Build keyword families around the core topic, synonyms, methods, systems, mechanisms, controversies, applications, and recent terms.
2. Search or ingest at least 200 topic-relevant records for full reviews.
3. Read available full texts where lawful and accessible for deep reviews.
4. Record source, query, inclusion logic, full-text count, unavailable-access boundary, and core evidence set.
5. Do not begin main drafting until this log exists, unless the user provides a verified corpus and explicitly asks to proceed.

The log can be a short QC note, table, or evidence registry. It must be concrete enough for another reviewer to see what was searched and what was actually read.

## 5-8 Theme Rule

After literature intake, condense the review into 5-8 synthesis themes.

Use this balance:

- Themes 1-3 or 1-4: broad, detailed coverage themes that define the field, measurement boundaries, historical shift, conceptual architecture, and major evidence layers.
- Later themes: deeper analysis of the core topic's features, characteristics, distinctive strengths, bottlenecks, exceptions, controversies, translation value, or viewpoint-level synthesis.

Each theme should include:

- theme title,
- one-sentence viewpoint,
- evidence base,
- core uncertainty or bottleneck,
- figure/table role if needed.

Do not confuse themes with headings. A theme can span more than one section, and one section can support part of a larger theme. The outline should still map every main section back to one of the 5-8 themes.
