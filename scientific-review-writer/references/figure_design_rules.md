# Scientific Figure Design Rules

Use these rules when planning or generating figures for a review manuscript.

## Visual Defaults

- White or very pale background.
- Readable labels.
- Consistent font sizes and line weights.
- restrained, accessible palette.
- Captions carry interpretation; in-figure text stays short.
- Avoid decorative effects that obscure data or mechanism.

## Publication Feel

Use figures to make the review easier to reason about:

- conceptual model,
- mechanism map,
- evidence hierarchy,
- exception taxonomy,
- method or validation pipeline,
- data-driven summary.

Avoid generic icons, unsupported molecular pathways, decorative gradients, and visual claims not present in the text.

## Final Checks

- Does the figure answer one clear question?
- Is it cited in the text?
- Does the caption explain what it synthesizes?
- Are labels legible after export?
- Does it match the actual user data or literature evidence?
- Are all file paths relative and stable?
