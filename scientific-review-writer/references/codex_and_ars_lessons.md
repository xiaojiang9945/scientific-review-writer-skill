# Lessons Adapted From External References

## Adapted From Academic Research Skills

- Use staged pipelines rather than one-shot drafting.
- Add checkpoint gates before major transitions.
- Keep a claim-evidence integrity pass separate from prose polish.
- Support optional style calibration from user-provided writing.
- Add writing-quality checks that improve clarity rather than disguising authorship.
- Treat figure verification as part of manuscript integrity.

## Adapted From Codex Best Practices

- Keep the skill scoped to one repeatable job.
- Put durable workflow rules in skill files and repository instructions.
- Use validation scripts when repeated errors are likely.
- Keep instructions practical and split detailed rules into reference files.
- Run checks and review diffs before accepting changes.
