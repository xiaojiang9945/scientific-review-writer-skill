# Review Workflow

Use this workflow for full plant-science review manuscripts and major revisions.

## Full Manuscript

1. Define the thesis in one sentence.
2. Define scope: organism, crop, tissue, process, time period, and application boundary.
3. Run the style-learning gate from at least 200 relevant high-level review records or approved exemplars.
4. Run the topic-learning gate from at least 200 topic-relevant literature records.
5. Build a core evidence set for close reading and citation.
6. Build a section map and length plan.
7. Build a figure map beside the section map.
8. Draft title options, abstract, keywords, main text, outlook, outstanding questions, figures, captions, and references.
9. Run five QC rounds.
10. Deliver Markdown and, when requested or feasible, DOCX/Word output.

## Revision

1. Diagnose whether the criticism is about scope, evidence, logic, language, formatting, or figures.
2. If the criticism exposes a missing scientific axis, run a targeted literature refresh before rewriting.
3. Revise the manuscript arc, not only the sentence where the issue was noticed.
4. Update figures, tables, captions, references, and outlook together when the thesis changes.
5. Run the five-round QC again for substantial revisions.

## Access Boundaries

Record what was actually searched and read. Acceptable sources include public metadata, abstracts, open full texts, DOI pages, publisher pages, NCBI/PubMed, Europe PMC, OpenAlex, institutional records, and user-provided files. Do not imply subscription-only full-text access unless it was actually available.
