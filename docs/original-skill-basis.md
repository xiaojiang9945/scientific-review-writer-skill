# Original Skill Basis

This repository is a generalized publication export of the existing local review-writing skill lineage.

## Included From The Original Skill

- The core review-manuscript workflow.
- Evidence-ranked claim discipline.
- Figure-rich manuscript standards.
- Citation and reference QC.
- DOCX build and audit scripts.
- Multi-round QC requirements.

## Generalized In This Version

- Removed plant-only trigger language.
- Renamed the skill to `scientific-review-writer`.
- Made English the default output language.
- Added cross-disciplinary review intake fields.
- Added style calibration and writing quality checks.
- Added staged pipeline and checkpoint language learned from academic-research workflow patterns.
- Set default full-review scale from the latest cytoskeleton/cell-division review.

## Excluded From The Publication Export

- Private project briefs.
- Large corpus TSV files that should be regenerated or supplied locally rather than redistributed.
- Raw full-text downloads.
- Local absolute paths.
- Account names, credentials, tokens, or private workspace notes.
