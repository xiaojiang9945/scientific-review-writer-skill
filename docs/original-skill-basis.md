# Original Skill Basis

This repository is a publication export of the existing local `plant-review-writer` skill, not a separate skill lineage.

## Included From The Original Skill

- Core `SKILL.md` workflow, revised for public release.
- General plant-review style rules.
- Review output standard.
- Review writing rules.
- Figure design rules.
- Citation QC checklist.
- Workspace-derived general lessons.
- Corpus summary.
- Reusable corpus collection, style analysis, DOCX build, and DOCX audit scripts.

## Excluded From The Publication Export

- Private project briefs.
- Large corpus TSV files that should be regenerated or supplied locally rather than redistributed.
- Raw full-text downloads.
- Local absolute paths.
- Account names, credentials, tokens, or private workspace notes.

## Added During This Release Preparation

- Nature-family reference-style guide.
- Five-round QC guide.
- Private demo based on the plant kinetochore review.
- Demo validation script.
- Release, security, use-policy, and privacy checklists.
